API Reference

URL Description Method
/api/product/get Returns all products GET
/api/product/get/9310015240638 Returns a specific product with barcode: 9310015240638 GET
/api/product/new?name=Cookies&brand=Arnotts&category=Biscuits&barcode=123456789 Creates a new product with barcode: 123456789 POST



/api/user/display User records for display GET
/api/user/get/224 Returns a specific user with id: 224 GET
/api/user/get Returns all users GET
/api/user/update/224?lname=chen Update last name with norman of the user with id 224 PUT
/api/user/add?fname=Tony&lname=Young&email=hghgh@gmail.com&password=oro Creates a new user with name Tony Young, with some info POST
