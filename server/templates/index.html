API Reference

URL Description Method

Product

/api/product/get Returns all products GET
/api/product/get/9310015240638 Returns a specific product with barcode: 9310015240638 GET
/api/product/new Creates a new product with barcode: 123456789 POST
/api/product/update Updates a product by the given form data PUT

USER

/api/user/display User records for display GET
/api/user/get/224 Returns a specific user with id: 224 GET
/api/user/get Returns all users GET
/api/user/getByUsername/adamp Returns user by given username GET
/api/user/getByEmail/adam@gmail.com Returns user by given email GET
/api/user/getBy/adam@gmail.com Returns user by given email OR username GET
/api/user/update Updates the user given by 'user_id' key in request form. Return updated user object or {"error": "user not found"}. PUT
/api/user/add Creates a new user with the data included in the POST form. Returns created user. POST
/api/user/delete/(user_id) Deletes the user given by 'user_id' key in request form. Return {"success": "user deleted"} or {"error": "user not found"}. DELETE
/api/user/deleteByEmail Deletes the user given by 'email' key in request form. Return {"success": "user deleted"} or {"error": "user not found"}. DELETE

Feedback

/api/feedback/get Returns all users' feedback GET
/api/feedback/get?224 Returns feedback that is from user with id: 224 GET
/api/feedback/new Creates a new feedback written from user with given id in the request body POST

Favourite

/api/favourite/get Returns all products that have been starred GET
/api/favourite/add Sets a product starred given by 'user_id' key and 'product_id' in request form.
Return {JSON of the added starred product} or {"status": "error", "message": "already a favourite"} or {"status": "error", "message": "missing parameter(s)"} 		POST
/api/favourite/remove Deletes a product given by 'user_id' key and 'product_id' in request form.
Return {"status": "success", "message": "favourite removed"} or {"status": "error", "message": "already a favourite"} or {"status": "error", "message": "missing parameter(s)"}		 POST
